ℹ️ How Selloship Courier Selection Works

Selloship does not have a courier listing API. Instead, courier selection works in two ways:

🤖 Auto Routing (Recommended)
Leave courierName and courierID blank in the waybill payload. Selloship automatically picks the best courier for the pincode.
🎯 Manual Override
Set courierName (e.g. Delhivery Fr) and courierID in the Mappings tab. These are sent in the waybill payload to force a specific courier.
✅ Setup Checklist
💡 To use Delhivery Fr or any specific courier:
  1. Go to Platform Couriers → create courier named "Delhivery Fr"
  2. Go to Courier Mappings → add mapping for it
  3. Set Selloship Courier Name = Delhivery Fr (exact name Selloship uses)
  4. Set Selloship Courier ID = the ID from your Selloship account manager
  5. When client ships with "Delhivery Fr", that name+ID goes in the waybill payload
